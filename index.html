La Web Moderna

HTML

(HyperText Markup Language): es el lenguaje estándar para crear la estructura y contenido de páginas web. Define elementos como encabezados, párrafos, imágenes y enlaces.

Ver más

CSS

CSS (Cascading Style Sheets): se utiliza para dar estilo y diseño a las páginas web creadas con HTML. Controla aspectos visuales como colores, fuentes, tamaños y disposición de los elementos.

Ver más

JavaScript

Es un lenguaje de programación que agrega interactividad a las páginas web. Permite crear efectos dinámicos, manipular el DOM y comunicarse con servidores.

Ver más

HTTP

HTTP (HyperText Transfer Protocol): es el protocolo de comunicación utilizado para transferir datos entre servidores web y navegadores. Es la base de la World Wide Web.
